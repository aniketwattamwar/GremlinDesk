Graph of the Gods Examples

                    
Click any query below to instantly load it into the input bar.

                    

                        
                        

                            
1. The Hercules Path

                            
Shows a rich visual path of Hercules battling monsters.

                            g.V().has('name', 'hercules').outE('battled').inV().path().toList()
                        


                        

                            
2. Find Hercules' Parents

                            
Traverses the 'father' and 'mother' edges, returning the parent vertices.

                            g.V().has('name', 'hercules').out('father', 'mother').toList()
                        


                        

                            
3. Pluto's Brothers

                            
Finds Pluto's brothers as graph vertices.

                            g.V().has('name', 'pluto').out('brother').toList()
                        


                        

                            
4. Who lives in Tartarus?

                            
Traverses incoming 'lives' edges to return the monster vertices.

                            g.V().has('name', 'tartarus').in('lives').toList()
                        


                        

                            
5. View All Edges

                            
Returns a sample of 20 raw edges in the graph.

                            g.E().limit(20).toList()
